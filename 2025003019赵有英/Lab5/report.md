# Android Jetpack Compose ArtSpace 艺术空间应用实验报告

## 一、实验基本信息
- 实验名称：基于 Jetpack Compose 的艺术作品展示应用开发
- 开发环境：Android Studio
- 开发语言：Kotlin
- UI 框架：Jetpack Compose (Material 3)
- 实验内容：完成艺术作品图片展示、信息展示、上下页切换功能

## 二、实验目的
1. 掌握 Jetpack Compose 声明式 UI 的基本开发方式与常用界面组件的使用。
2. 学会使用状态管理实现界面数据的动态更新与作品切换。
3. 按照效果图要求实现统一的界面尺寸、间距、颜色与排版样式。
4. 实现图片区域、信息展示区域、按钮控制区域的合理布局。
5. 掌握图片资源管理、界面交互逻辑编写及简单样式美化。

## 三、实验原理
1. Jetpack Compose 采用声明式界面构建方式，通过组合函数搭建 UI，无需传统 XML 布局。
2. 使用 remember 与状态变量保存当前显示的作品索引，实现点击切换后界面自动刷新。
3. 通过 Column、Row 等布局组件实现垂直与水平排列，完成整体界面结构划分。
4. 使用图片组件加载本地资源图片，配合卡片、阴影、圆角等样式提升界面美观度。
5. 通过按钮点击事件修改状态索引，实现艺术作品的循环切换。

## 四、实验内容与步骤
1. 创建新项目并配置 Jetpack Compose 环境，确保项目能够正常运行与预览。
2. 将艺术作品图片资源添加到项目中，整理并删除无用图片资源，避免引用错误。
3. 定义数据结构存储每幅作品的信息，包括图片、标题、作者与创作年份。
4. 搭建主界面布局，将界面分为图片展示区、作品信息区、按钮控制区三部分。
5. 实现图片区域卡片样式，设置阴影、圆角、内边距，使其与效果图保持一致。
6. 编写信息展示区域，设置文字大小、颜色、对齐方式与背景样式。
7. 实现 Previous 和 Next 按钮，设置按钮大小、颜色与圆角样式。
8. 添加状态管理逻辑，实现点击按钮切换上一张、下一张图片，并支持循环切换。
9. 调整整体间距、宽度、高度等细节，使最终界面与效果图完全匹配。
10. 运行程序测试功能，检查切换是否正常、文字是否清晰、布局是否整齐。

## 五、实验结果
1. 应用成功运行，界面整体风格与效果图一致，尺寸、颜色、间距均符合要求。
2. 图片区域以卡片形式展示，带有阴影和圆角效果，图片显示完整且比例正常。
3. 作品信息区域文字清晰排列，标题与作者信息区分明显，背景颜色柔和统一。
4. 两个按钮宽度均分、大小统一，点击时可以正常切换不同艺术作品。
5. 实现循环切换功能，从第一张可向前切到最后一张，从最后一张可向后切到第一张。
6. 程序运行稳定，界面切换流畅，无报错、无布局错乱现象。

## 六、问题与解决方法
1. 问题：图片引用错误导致程序无法运行。
   解决：核对资源文件名，删除无用图片，确保代码中引用的资源真实存在。
2. 问题：界面大小与效果图不一致。
   解决：调整宽度比例、内边距、间距和文字大小，使其视觉效果统一。
3. 问题：按钮切换不能循环。
   解决：增加边界判断逻辑，到达第一张或最后一张时跳转到对应位置。
4. 问题：文字排版不整齐。
   解决：设置对齐方式、字号粗细和间距，使信息展示更规范。

## 七、实验总结
本次实验通过 Jetpack Compose 完成了 ArtSpace 艺术作品展示应用，实现了图片浏览、信息展示、上下页切换等功能。实验过程中掌握了声明式 UI 的基本开发思路，学会了状态管理、界面布局、资源引用和交互逻辑实现方法。通过不断调整界面细节，使最终效果与效果图保持一致，提升了界面调试与样式美化的能力。整体实验达到预期目标，为后续 Android 应用开发打下了良好基础。
